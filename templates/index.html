Quick Start

Say you want have a website where you want to list some dummy users to see whether your css holds up.

All you need to do is to create an ajax request to the url https://openapi.heroku.com/genericdata/users/100

                        fetch('https://openapi.heroku.com/genericdata/users/100')
                        .then((res)=>res.json())
                        .then(data=>callback(data))
                        .catch(e=>{
                            Messages.error("Could not fetch data from endpoint")
                            console.error(e)
                            console.log("Could not fetch from endpoint https://openapi.heroku.com/genericdata/users/100")
                        })

Some other genenric data include

  • Products
  • Reviews
  • Songs thumbnails
  • Video thumbnails

All the generic data can be accessed via the url https://openapi.heroku.com/genericdata/[genertic_data_name]/[number]

                        [generic_data_name] -> this is the name of the generic data you want.
                        It should be lower case. You can refer to the list above for the available generic data samples
                        [number]-> this is self explanatory but for documentation purposes it is the number of items of the generic data you need.
